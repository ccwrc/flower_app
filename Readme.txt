"Do wykonania jest prosta aplikacja w PHP.


 Poniżej przedstawiam główne oraz jedyne wytyczne.

 - powinna być odpalana z poziomu CLI.

 - powinna za każdym uruchomieniem pobrać 3 losowe zdjęcia z strony sklep.swiatkwiatow.pl

 - przy każdym pobraniu tych zdjęć powinno odnotować ten fakt tak, że przy ponownym uruchomieniu tej aplikacji powinno pomijać odnalezione zdjęcia

 - w przypadku wystąpienia jakiegokolwiek erroru informacja o tym powinna być zapisana w pliku log.err


 Kwestia przekazania aplikacji do oceny jest dowolna :) "
